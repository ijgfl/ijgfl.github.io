# ijgfl.github.io
Um site de portfólio para a av2 de introdução à computação nele estão contidas 3 páginas: index, projetos e now. No index tem uma bio pequena, em projetos estão contidos os meus projetos da faculdade até agora, e em now estão contidas as coisas que eu estou fazendo no momento. Esse projeto de portfólio utiliza js, css e html, js é utilizado para contar o número de visitas ao site, estruturar a página de projetos e criar o efeito das estrelas passando.
### IMPORTANTE: após múltiplas tentativas de realizar o efeito em "warp.js" sem êxito, eu recorri ao Claude para conseguir o visual das estrelas passando, como só é um efeito visual sem nenhuma importância, eu torço para que não seja um problema.
